คำถามที่พบบ่อย

GPA คำนวณจากผลรวมของ (เกรด × หน่วยกิต) ของทุกวิชา หารด้วยผลรวมหน่วยกิตทั้งหมด เช่น วิชา A เกรด 4.0 หน่วยกิต 3 และวิชา B เกรด 3.0 หน่วยกิต 3 ได้ GPA = (4×3 + 3×3) ÷ (3+3) = 3.50
GPA คือเกรดเฉลี่ยของเทอมนั้นๆ ส่วน GPAX คือเกรดเฉลี่ยสะสมตลอดหลักสูตร คำนวณจากทุกวิชาทุกเทอมรวมกัน GPAX จึงเปลี่ยนช้ากว่า GPA รายเทอม
A = 4.0, B+ = 3.5, B = 3.0, C+ = 2.5, C = 2.0, D+ = 1.5, D = 1.0, F = 0.0 มหาวิทยาลัยส่วนใหญ่ในไทยใช้ระบบนี้ แต่บางแห่งอาจมี S (Satisfactory) หรือ U (Unsatisfactory) สำหรับวิชาพลศึกษาหรือกิจกรรมที่ไม่นับ GPA
